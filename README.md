Customer Shopping Behavior Analysis

1. Project Overview

This project analyzes customer shopping behavior using Python, MySQL, and Power BI to uncover meaningful business insights. It demonstrates a complete data analytics workflow, from raw data preprocessing to interactive dashboard creation and business reporting.

The project focuses on cleaning and preparing customer data, performing SQL-based analysis, building visual dashboards, and communicating insights through a professional report and presentation.

The dataset contains customer shopping information, including:

4. Project Workflow

Step 1: Data Loading


Imported the dataset into Jupyter Notebook using Pandas.
Performed an initial inspection of the dataset.


Step 2: Exploratory Data Analysis (EDA)


Explored dataset structure
Generated descriptive statistics
Checked missing values
Examined data quality


Step 3: Data Cleaning & Feature Engineering


Handled missing values
Standardized column names
Renamed columns for consistency
Created new features such as:

Age Group
Purchase Frequency (Days)



Removed redundant columns


Step 4: SQL Analysis (MySQL)

Imported the cleaned dataset into MySQL and answered business questions such as:


Total revenue by gender
Customer spending patterns
Highest-rated products
Shipping type comparison
Subscriber vs non-subscriber analysis
Discount usage analysis
Customer segmentation
Product ranking within categories
Repeat customer analysis
Revenue contribution by age group


Step 5: Power BI Dashboard

Designed an interactive dashboard to visualize business insights using charts, KPIs, slicers, and filters.

Step 6: Report & Presentation


Prepared a business report summarizing key findings.
Created a presentation using Gamma to communicate insights.
